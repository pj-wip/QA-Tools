# QA Utility Toolkit 🛠️
A browser-based utility tool built with HTML, CSS, and JavaScript to streamline day-to-day QA workflows — no installations, no dependencies, just open and use.

---

## Features:

### 💬 Comment Template Manager
Store and reuse frequently used defect comments so you don't have to retype the same thing over and over. Manage your template library directly in the browser and paste ready-made comments into your bug reports in seconds.

### ✅ Interactive Testing Checklist
Work through test cases with an interactive checklist built for QA sessions. Once you're done, capture your completed checklist as a screenshot — it downloads automatically as an image, ready to attach to a report or ticket.
